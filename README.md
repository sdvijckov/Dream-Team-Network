## 📚 Подпроекты

### 📊 [datasets](datasets/README.md)
Система подготовки, разметки и векторизации текстовых данных.

**Назначение:**
- Загрузка текстов из .txt, .md, .docx
- Автоматическая очистка и нормализация
- Разметка (персонажи, темы, эмоции)
- Создание векторных представлений для семантического поиска
- Подготовка датасетов для обучения нейросетей и LoRA адаптеров

**Быстрый старт:**
```bash
cd datasets
install.bat
python cli.py process
```

📖 [Полная документация](datasets/README.md)

### 🤖 models (в разработке)
Обученные LoRA адаптеры для агентов проекта.

**Планируемые адаптеры:**
- `oleg_writer` — авторский стиль Олега Сдвижкова
- `sheldon_influencer` — англоязычный контент
- `genrikh_teacher` — мудрые диалоги и философия
- `smelchak_teacher` — энергичные практико-ориентированные ответы

---

### 🌐 api (в разработке)
Серверная часть для интеграции с Dream Team и другими проектами.

**Функционал:**
- REST API для доступа к обработанным данным
- Семантический поиск через векторную базу
- Управление агентами и профилями
- Интеграция с RAG системами

## 🛠️ Установка и запуск

### Для каждого подпроекта:

1. Перейти в папку подпроекта
2. Запустить `install.bat` (если есть)
3. Следовать инструкциям в README

### Общий проект:

```bash
# Клонировать репозиторий
git clone <url>

# Перейти в папку проекта
cd "Dream Team Network"

# Выбрать подпроект для работы
cd datasets
install.bat
```
